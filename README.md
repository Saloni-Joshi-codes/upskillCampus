# Blog CMS

## Project Overview
Blog CMS is a Full Stack Content Management System that allows users to create, view, update, and delete blog posts. It is built using Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript.

## Features
- Create Blog
- View Blogs
- Edit Blog
- Delete Blog
- MongoDB Database Integration
- REST API using Express.js

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- MongoDB
- Mongoose
